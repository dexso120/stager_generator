Self Extracting Command (Powershell)

Reference:
https://mp.weixin.qq.com/s/nJpqvXCYV3ZdvNgYGrG4ow

A powershell command that tries to locate itself (a zip file) from the user's directory using where.exe. Then it extracts itself using Expand-Archive.

powershell 
$selfLocation=(where.exe /r $env:USERPROFILE '<filename>') | select -First 1;
$selfLocation=$selfLocation.Trim();
Expand-Archive $selfLocation -D $env:APPDATA\<folder>;
